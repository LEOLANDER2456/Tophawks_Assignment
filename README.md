 Tophawks AI Internship Assignment

This project was developed as part of the Tophawks AI Internship screening assignment.

It demonstrates practical implementation of AI-driven solutions for lead scoring, data cleaning, and workflow automation using real-world datasets.

---

## 🚀 Features

- Lead Scoring System to prioritize high-value prospects  
- Data Wrangling Pipeline to clean messy real-world data  
- Workflow Automation for generating follow-up emails  
- Mini AI Tool for automated lead analysis

## 🤖 Mini AI Tool

The mini AI tool processes lead data, assigns scores based on engagement signals, and ranks leads to identify high-value prospects.

---

## ⚙️ How to Run

Clone the repository:

git clone https://github.com/LEOLANDER2456/Tophawks_Assignment.git

Navigate to the folder:

cd Tophawks_Assignment


Run the scripts:

python lead_scoring.py
python messy_data.py
python workflow_email.py
python app.py
